# 🌐 CampusCircle  

> A scalable university networking platform built with the MERN stack.  
> Designed to enable structured communities, real-time interaction, and role-based governance within university ecosystems.

🚧 **Status: Actively Under Development**

---

## 📌 Overview  

CampusCircle is a full-stack university networking platform focused on:

- Structured community discussions  
- Secure role-based access control  
- Real-time interaction capabilities  
- AI-assisted profile enhancements  
- Event synchronization features  

The goal is to engineer a production-grade campus platform with scalability and security in mind.

---

# ✅ Completed Modules  

## 1️⃣ Community Module  

- Create and manage university groups  
- Role-Based Access Control (Admin / Moderator / Member)  
- Approval-based post moderation system  
- Queued Post workflow for content validation  
- Secure high-stakes actions (admin delegation, removals)  

This module focuses on governance, structured discussions, and scalable community management.

---

## 2️⃣ Profile Module  

- Structured academic profile system  
- AI-assisted professional bio generation (Gemini API integration)  
- Editable and preview-based profile workflow  
- Clean UI flow with user-controlled refinement  

This module is built to help students present themselves professionally within the campus ecosystem.
